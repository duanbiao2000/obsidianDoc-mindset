好的，要列出当前仓库中所有标签并根据使用次数降序排列，可以使用 FLATTEN 和 GROUP BY 结合的方式。

以下是相应的 Dataview 命令：


```dataview
TABLE length(rows) AS "使用次数"
FROM ""
FLATTEN file.tags AS tag
WHERE tag != null
GROUP BY tag AS "标签"
SORT length(rows) asc
limit 20
```
